Library System

Overview

Edmon Low Library, a library system encompasses several essential features. The library operates daily from 9 am to 5 pm and is situated on Oklahoma State University campus with designated address. It supports functionalities for adding books, borrowing and returning them, and checking their availability.

To achieve this, a class named EdmonLowLibrary should be created to encapsulate the library's attributes and behaviors. This class must include instance fields and methods to manage the library operations effectively.

Library Features

Edmon Low library has the following features:

  • Opening Hours: 9 am to 5 pm
  • Address: 216 Library, Oklahoma State University, Stillwater, OK 74078
  • Operations:
    • Books can be added to the library.
    • Books can be borrowed from and returned (added back) to the library.
    • The availability of books can be checked.

Implementation Requirements

  • A class named EdmonLowLibrary should be created to represent the library system.
  • Implement all instance fields and methods required to manage the library system.
  • Implement a constructor to initialize the instance fields (attributes).
  • Implement an instance method to display the library's opening hours and address.
  • Implement each library operation as an instance method.
